# Визитка и портфолио

Одностраничный сайт: персональная визитка (hero-блок) и портфолио с контактами.

## Как открыть

1. Откройте папку `portfolio-site` в проводнике.
2. Дважды щёлкните по `index.html` — страница откроется в браузере.

Или откройте файл через Cursor: правый клик по `index.html` → «Open with Live Server» (если установлено расширение).

## Что настроить под себя

- **index.html**
  - Имя, должность, слоган: блок `.business-card` (Ваше Имя, Специалист/Дизайнер…).
  - Email, телефон, Telegram: ссылки в `.card-contacts` и в секции «Связаться».
  - Текст «Обо мне» и список навыков в секции `#about`.
  - Проекты: замените заглушки в `.project-card` на реальные названия, описания и ссылки; при желании замените `.project-placeholder` на тег `<img>` с превью.

- **Аватар**
  - Сейчас показываются инициалы «ФИ» в круге. Чтобы вставить фото: замените блок `.avatar-placeholder` на `<img src="ваш-файл.jpg" alt="Фото" class="avatar-img">` и в CSS задайте для `.avatar-img` размеры и `border-radius: 50%`.

- **Цвета и стиль**
  - В `styles.css` в начале файла в блоке `:root` можно поменять `--accent` (акцентный цвет), `--bg`, `--text` и остальные переменные.

## Структура

- `index.html` — разметка (визитка, обо мне, портфолио, контакты).
- `styles.css` — оформление и адаптив.
- `script.js` — год в подвале и лёгкая анимация появления карточек при скролле.

Сайт адаптирован под мобильные устройства.
